How to use Parallax to create a trendy scrolling effect

CSS Animation Tutorial

Parallax scrolling is a popular web site trend. The background content (example; an image) is moved at a different speed than the foreground content while scrolling.

While I intend to make this tutorial as simple as possible, I am also assuming you are familiar with HTML & CSS.

Step 1: Write the HTML

Inside of your .html document, create a container element (e.g. <div>) and add a class.

HTML <!-- Container element -->
<div class="parallax"></div>

Step 2: Write the CSS

You now have the container element. Let's write some CSS to it.

In your .css file. Insert the image you want, using background-image:

Add a min-height: to the container, and set a specific height using either px or %.

Now we will create the scrolling effect,
Add background-attachment: fixed;, background-position: center;, background-repeat: no-repeat;, background-size: cover; to the container.

All done!

The complete CSS code for the Parallax effect should look like this (ignore the comments '/* */' if you don't need them),

CSS .parallax {
/* The image used */
background-image: url("img_parallax.jpg");
/* Set a specific height */
min-height: 500px;
/* Create the parallax scrolling effect */
background-attachment: fixed;
background-position: center;
background-repeat: no-repeat;
background-size: cover;
}
